# Ping Pong Tracker

A React application for keeping track of score in ping pong.
Currently running on: https://ping-pong-tracker-eight.vercel.app/

---

## Features

- Keep score by pressing either player's side of the screen.
- Intended for Mobile Use.
- Yellow number indicates who is up to serve.
- Change score to win and serving rules in settings.

## Installation

### Clone the repo:

```
git clone https://github.com/llfisher932/PingPongTracker
cd PingPongTracker
```

### Install dependencies:

```
npm install
```

### Running the app:

```
npm run dev
```
